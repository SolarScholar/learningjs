Day 1

Variables

Variables are containers of data.

Variables can be declared in four ways:

  • Using let
  • Using const
  • Using var(Not Recommended)
  • Automaticaly(Not Recommended)

Example using let:

let x = 5;
let y = 6;
let z = x + y;

Examples using const:

const x = 5;
const y = 6;
const z = x + y;

Variables identifiers

The rules for constructing name(identifiers):

  • Names can contain letters, digits, underscores, and dollar signs.
  • Names must begin with a letter, a $ sign or an underscore (_).
  • Names are case sensitive (X is different from x).
  • Reserved words (JavaScript keywords) cannot be used as names.

Numbers aren't allowed as the first character names

Datatypes

  • String - a text of characters eclosed in quotes
  • Number - a number representing a mathematical value
  • Bigint - a number representing a large integer
  • Boolean - a data type representing true or false
  • Object - a collection of key-value pairs of data
  • Undefined - a primitive variable with no assigned value
  • Null - a primitive value representing object adsence
  • Symbol - a unique and primitive identifier
